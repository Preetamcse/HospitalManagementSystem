# ðŸŒŸ Introduction
**MediTrack** is a simple and efficient **C++ console-based Hospital Management System**.  
It allows admitting patients, storing their details, and discharging them by updating records using file handling.  
This project is perfect for learning C++ classes, file operations, and basic data management logic.

---

# ðŸ”¥ Features
- Admit new patients with name, address, and blood group  
- Automatically store patient data in a text file  
- Discharge patients by removing their record from the file  
- Clean class-based structure for easy understanding  
- Maintains updated data using file rewriting  
- Console interface using system commands for smooth interaction
